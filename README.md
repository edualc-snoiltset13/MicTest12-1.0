## Calculator (Flask)

Basic four-operation calculator with a custom recursive-descent parser.

### Run

```
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Open http://127.0.0.1:5000

Supports `+ - * /`, parentheses, unary minus, floats, and scientific notation
(e.g. `1e3`, `-2.5`).
